# Installation Guide

## Prerequisites
- Docker 20.10+
- Docker Compose 2.0+
- 4GB RAM minimum
- 50GB disk space

## Quick Install
```bash
git clone https://github.com/AlexanderMonsanto/reifenhaeuser-nixos-elk.git
cd reifenhaeuser-nixos-elk
make setup
make up
```
